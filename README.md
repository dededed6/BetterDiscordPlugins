# BetterDiscord Plugins

A collection of privacy-focused BetterDiscord plugins.

## Plugins

### BlockTrack
Block Discord tracking and analytics events, including science events, Sentry error reports, telemetry, A/B experiments, typing indicators, read receipts, activity status, and WebRTC IP leaks.

### CleanUrls
Remove tracking parameters from URLs using ClearURLs rules. Automatically cleans URLs from Google, Amazon, Facebook, and more.

### MetadataRemover
Remove metadata from files (images, audio, documents) before uploading to Discord. Supports JPEG, PNG, WebP, GIF, TIFF, MP3, FLAC, AAC, and PDF formats.

## Installation

Download the plugins from the `Plugins` folder and place them in your BetterDiscord plugins directory.
